# 引き継ぎ：労働衛生コンサルタント口述試験対策アプリのGitHub Pagesデプロイ

## 📋 概要

労働衛生コンサルタント（保健衛生区分）の口述試験対策用チャットボットアプリを作成しました。GitHub Pagesにデプロイして、スマホやPCからいつでもアクセスできるようにしたいです。

---

## 🎯 やりたいこと

1. GitHubリポジトリを作成
2. GitHub Pagesを有効化してWebアプリとして公開
3. `https://ユーザー名.github.io/リポジトリ名/` でアクセスできるようにする

---

## 📁 ファイル構成

ZIPファイル（github-pages-v2.zip）を展開すると以下の構成になっています：

```
github-pages-v2/
├── index.html          # アプリ本体（React + Tailwind CSS）
├── data/
│   ├── questions.md    # 本編の問題（予想問題集）
│   └── followup.md     # 追加問題編（追加質問回答集）
└── README.md           # 説明書
```

---

## 🎮 アプリのモード

| モード | 説明 | データソース |
|--------|------|--------------|
| **本編** | 基本→詳細→応用の3段階構成 | `data/questions.md` + 手動追加 |
| **追加問題編** | 補足的な知識を補強 | `data/followup.md` + 手動追加 |
| **復習フォルダ** | 5点以下の問題を復習 | localStorage |

## 📝 問題の追加方法

### ファイルから追加（推奨）
- **本編**: `data/questions.md` を編集してpush
- **追加問題編**: `data/followup.md` を編集してpush

### アプリ内で追加
- 「問題を管理・追加」から手動追加可能
- 手動追加した問題はlocalStorageに保存
- ファイルから読み込んだ問題と統合して出題

---

## 🔧 技術仕様

- **フロントエンド**: React 18（CDN経由）、Tailwind CSS（CDN経由）
- **採点機能**: Anthropic API（Claude）を使用
- **データ保存**: localStorage（カスタム問題・復習フォルダ用）
- **問題データ**: マークダウンファイルをfetchで読み込み

---

## ✅ アプリの機能

| 機能 | 説明 |
|------|------|
| モード選択 | 本編・追加問題編・カスタム問題から選択 |
| ランダム出題 | 5テーマをランダムに選択 |
| AI採点 | 10点満点で採点、講評付き |
| 持ち帰り機能 | 答えられない問題は2回まで持ち帰り可能 |
| 復習フォルダ | 5点以下の問題を自動保存、後から復習可能 |
| 補足質問 | 本編で5点以下の場合、追加質問に答えるチャンスあり |
| 合格判定 | 回答したテーマの60%以上で合格 |
